# Airline Simulation

一款连接现实世界实时数据的航空模拟运营网页游戏。

## 快速开始

### 前置要求
- Node.js 18+
- Cursor IDE (推荐) 或 VS Code
- Supabase账号 (免费)
- GitHub账号

### 技术栈
- 前端: React 18 + Ant Design 5 + Leaflet
- 后端: Supabase (PostgreSQL + Edge Functions + Realtime)
- 算法: Python + Google OR-Tools
- 数据抓取: Make.com / n8n

### 项目结构
请阅读 docs/ARCHITECTURE.md 了解完整架构设计。
请阅读 docs/DEVELOPMENT_PROTOCOL.md 了解开发流程规范。

### 开发流程
1. 所有新想法先写入 ARCHITECTURE.md 的"想法池"
2. 经评估后移入正式架构
3. 按Phase顺序逐步开发
4. 每个功能完成后 git commit
